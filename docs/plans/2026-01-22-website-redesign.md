## 2. Lead Magnet 系統設計

### 2.1 代謝年齡測驗

**入口位置**：
- 首頁 Hero 主 CTA
- 首頁底部 CTA 區塊
- 每篇部落格文章底部
- 服務頁面側邊欄

**測驗流程**：
```
[開始] → 7 題問卷 → [輸入 Email] → [查看結果]
                                      ↓
                              寄送完整報告 + 影片推薦
```

**問卷題目設計（7 題）**：
1. 你的年齡區間？（30-40 / 40-50 / 50-60 / 60+）
2. 餐後是否容易感到昏沉想睡？（常常 / 偶爾 / 很少）
3. 是否有三高問題？（血壓 / 血糖 / 血脂 / 都沒有）
4. 目前的運動頻率？（幾乎沒有 / 每週 1-2 次 / 每週 3+ 次）
5. 睡眠品質如何？（常失眠 / 淺眠易醒 / 睡得不錯）
6. 是否嘗試過減重但反覆失敗？（是 / 否）
7. 最困擾你的健康問題是？（體重 / 疲勞 / 三高 / 睡眠 / 其他）

**結果分類（3 種）**：
- 🟢 代謝年輕型（+0~5 歲）→ 推薦：維持型內容
- 🟡 代謝警示型（+5~15 歲）→ 推薦：入門改善內容
- 🔴 代謝老化型（+15 歲以上）→ 推薦：預約諮詢

**結構**：

```
│  • 運動不足導致線粒體功能下降                                 │
│                                                             │
│  📧 完整報告已寄送到你的信箱                                  │
│                                                             │
│  🎬 推薦你先看這支影片：                                      │
```

## 3. 服務頁面與預約流程優化

### 3.1 諮詢預約流程重設計

**新流程**：
```
[預約諮詢 CTA]
      ↓
[Surveycake/Google 表單]（預約問卷）
      ↓
[表單結尾顯示下一步指示]
      ↓
[博士收到通知後，手動或自動回覆確認時間]
```

### 3.2 預約問卷設計（Surveycake 或 Google 表單）

**問卷欄位（6 題）**：

1. 你的姓名與稱呼？（文字輸入）
2. Email（文字輸入，必填）
3. 你目前最困擾的健康問題？（複選：體重 / 三高 / 疲勞 / 睡眠 / 其他）
4. 是否有進行中的醫療治療？（是/否，若是請簡述）
5. 你希望在 90 天後達成什麼目標？（文字輸入）
6. 方便諮詢的時段？（複選：平日白天 / 平日晚上 / 週末）

**表單結尾感謝頁**：
```
┌─────────────────────────────────────────────────────────────┐
│  ✅ 已收到你的預約申請！                                      │
│                                                             │
│  阿杰博士會在 1-2 個工作天內，透過 Email 與你確認諮詢時間。    │
│                                                             │
│  在等待期間，推薦你先觀看：                                   │
│  🎬 [什麼是代謝靈活性？] ← YouTube 連結                       │
│                                                             │
│  [返回首頁]                                                  │
```

## 5. 視覺設計系統升級

### 5.1 色彩系統優化

**維持現有基底，增加層次**：

| 用途 | 現有 | 調整 |
|------|------|------|
| 主色（CTA、強調） | `#d97706` 琥珀色 | 維持 |
| 輔色（健康、成功） | `#15803d` 綠色 | 維持 |
| 科學感強調（新增） | - | `#1e40af` 深藍色 |
| 背景色 | `#fffbeb` 暖白 | 維持 |
| 深色文字 | `rgb(28,25,23)` | 維持 |

**新增深藍色用途**：
- 數據視覺化圖表
- 科學證據區塊背景
- 與琥珀色形成「科學 + 溫暖」的視覺平衡

### 5.2 字體與排版

**維持 Atkinson 字體**，調整層級：

| 元素 | 現有 | 調整 |
|------|------|------|
| Hero 主標 | 3.052em | 3.5em，加粗 |
| 區塊標題 | 2.441em | 維持 |
| 卡片標題 | 1.953em | 維持 |
| 內文 | 20px | 維持 |
| 小標/標籤 | - | 新增 14px 大寫字母 |

### 5.3 元件設計升級

- **按鈕**：加入 hover 漸層與微動畫
- **卡片**：維持綠色頂邊框，加入 hover 陰影放大
- **表單**：圓角輸入框、focus 狀態琥珀色邊框

## 6. 技術實作規劃

### 6.1 技術架構維持

**維持現有 Astro 框架**，新增以下整合：

| 功能 | 技術方案 |

| 類型 | 項目 | 說明 |
